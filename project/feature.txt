# 기능 개발 실습 파일

이 파일은 각 브랜치에서 서로 다른 기능을 개발하는 실습에 사용됩니다.

## 현재 구현된 기능
- 기본 설정 완료
- 초기 파일 구조 생성
- 🚨 사용자 로그인 기능 (main에서 직접 작업!)

## 앞으로 추가될 기능
- 데이터 처리 기능
- UI 컴포넌트
- API 연동

## 🚨 실수 상황: main 브랜치에서 직접 작업
이 커밋은 실수로 main 브랜치에서 직접 작업한 상황을 시뮬레이션합니다.
실제로는 feature 브랜치를 만들어서 작업했어야 합니다!

### 추가된 기능
- 사용자 로그인 폼 UI 추가
- 기본 인증 로직 구현
- 세션 관리 기능
